# Cтруктура данных

Структура данных — это способ хранения данных.

Например:

- список (`list`) хранит элементы подряд;
- словарь (`dict`) хранит пары ключ-значение;
- множество (`set`) хранит уникальные элементы;
- стек хранит элементы по принципу последний пришёл — первый ушёл;
- очередь хранит элементы по принципу первый пришёл — первый ушёл.

## Stack (Стек)

Стек — это базовая структура данных, которая управляет порядком обработки элементов.

В Python нет отдельного встроенного типа `Stack`. Поэтому стек обычно реализуют через список (`list`). А у списка уже свои методы.

`stack = []`

- Положить элемент в стек.
  В теории: `push(10)`

  В Python: `stack.append(10)`

  Потому что `append()` добавляет элемент в конец списка, а конец списка мы считаем верхушкой стека.

# 2. Стек (Stack)

## Принцип работы

Стек работает по принципу:

> **LIFO (Last In — First Out)**  
> Последний пришёл — первый вышел

Аналогия: стопка тарелок.

---

## Основные операции (теория)

| Операция | Описание |
|----------|----------|
| push(x) | добавить элемент |
| pop() | удалить верхний элемент |
| top() / peek() | посмотреть верхний элемент |
| empty() | проверить пуст ли стек |

---

## Реализация в Python

В Python нет отдельного типа Stack, поэтому используют `list`.

| Стек (теория) | Python |
|----------------|--------|
| push(x) | append(x) |
| pop() | pop() |
| top() | stack[-1] |

---

## Пример

```python
stack = []

stack.append(10)  # push
stack.append(20)
stack.append(30)

print(stack[-1])  # top -> 30

print(stack.pop())  # pop -> 30
print(stack)  # [10, 20]
