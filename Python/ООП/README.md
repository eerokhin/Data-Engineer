## 1. Класс — это "чертёж".

Класс — это инструкция по созданию объекта (это шаблон или чертёж). Например, описание автомобиля:

```python
class Car:
    pass
```

Сам по себе класс не является конкретной машиной. Он просто говорит, какими будут объекты.

## 2. Объект (Экземпляр класса (instance)). 

Объект (экземпляр класса) — это конкретная сущность, созданная на основе класса, которая хранит свои данные (атрибуты) и может выполнять действия (методы), определённые в классе.

Создадим экземпляр:

```python
class Car:
    pass

car1 = Car()
car2 = Car()
```

Здесь:

`Car` — класс

`car1` — экземпляр класса

`car2` — экземпляр класса

В памяти это два разных объекта.

Проверить можно через id():

```python
print(id(car1)) #139049902988592
print(id(car2)) #139049902991184
```
Адреса будут разными.

## 3. Атрибуты (attributes).

Атрибуты — это данные, которые хранятся внутри объекта или класса.

Например:

```python
class Car:
    def __init__(self, brand, color):
        self.brand = brand
        self.color = color

car1 = Car("BMW", "black")
```

Теперь у объекта есть атрибуты:

```python
print(car1.brand)  # BMW
print(car1.color)  # black
```

Здесь:

`brand` — атрибут
`color` — атрибут

## 4. Атрибуты экземпляра.

Создаются через `self`.

```python
class Person:
    def __init__(self, name):
        self.name = name

p1 = Person("Анна")
p2 = Person("Иван")
```

У каждого объекта своё значение:

```python
print(p1.name)  # Анна
print(p2.name)  # Иван
```

Схематично:

```python
p1
 └─ name = "Анна"

p2
 └─ name = "Иван"
```

## 5. Атрибуты класса.

Создаются внутри класса, но вне методов.

```python
class Person:
    species = "Human"

p1 = Person()
p2 = Person()
```

Доступны всем объектам:

```python
print(p1.species)  # Human
print(p2.species)  # Human
```

Схематично:

```python
Person
 └─ species = "Human"

p1 ─┐
p2 ─┘ используют один и тот же атрибут
```

## 6. Разница между атрибутом класса и экземпляра.

```python
class Person:
    species = "Human"      # атрибут класса

    def __init__(self, name):
        self.name = name   # атрибут экземпляра

p1 = Person("Анна")
p2 = Person("Иван")
```

Получаем:

```python
print(p1.species)  # Human
print(p2.species)  # Human

print(p1.name)     # Анна
print(p2.name)     # Иван
```


## 7. Как Python ищет атрибут

Когда выполняется:

```python
p1.name
```

Python ищет:

1. В самом объекте `p1`
2. Если не нашёл — в классе `Person`
3. Если не нашёл — в родительских классах

Например:

```python
class Person:
    country = "Russia"

p1 = Person()

print(p1.country)
```

В объекте `p1` атрибута `country` нет, поэтому Python берёт его из класса.


## 8. Короткое определение. 

Класс — шаблон для создания объектов.

Экземпляр класса — конкретный объект, созданный по этому шаблону.

Атрибут — переменная, принадлежащая объекту или классу.

Пример:

```python
class Dog:
    species = "Dog"          # атрибут класса

    def __init__(self, name):
        self.name = name     # атрибут экземпляра

dog = Dog("Шарик")           # экземпляр класса
```

Здесь:

- `Dog` — класс
- `dog` — экземпляр класса
- `species` — атрибут класса
- `name` — атрибут экземпляра.


## 9. Метод (Method).

Метод — это функция, которая принадлежит классу.

```python
class Dog:
    def bark(self):
        print("Гав")

dog = Dog()
dog.bark() #Гав
```

## 10. Что такое self.

`self` — это ссылка на текущий экземпляр объекта.
